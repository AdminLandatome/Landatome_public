\(h\) est donc dérivable sur \(\mathbb{R}\backslash\lbrace{-\frac{9}{10}}\rbrace\) et pour tout \(x\in\mathbb{R}\backslash\lbrace{-\frac{9}{10}}\rbrace\),

\(h'(x)=\frac{-9(10x+9) -10(-9x+2)}{(10x+9)^2}\)

\[\forall x \in\mathbb{R}\backslash\lbrace{-\frac{9}{10}}\rbrace,\;\;\;h'(x)=\frac{-101}{(10x+9)^2}\]
