Donner l'expression de \(h'(x)\)

De plus, le dénominateur ne s'annule pas sur \(\mathbb{R}\backslash\lbrace{-\dfrac{1}{5}}\rbrace\).
\(h\) est donc dérivable sur \(\mathbb{R}\) et pour tout \(x\in\mathbb{R},\)

ou encore :
\(h'(x)=\frac{[5(-2x-10)+2]e^{5x-9}}{(-2x-10)^2}\)
