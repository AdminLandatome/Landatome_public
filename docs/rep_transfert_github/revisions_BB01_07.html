\(g\) est donc dérivable sur \(\mathbb{R}\backslash\lbrace{-4}\rbrace\) et pour tout \(x\in\mathbb{R}\backslash\lbrace{-4}\rbrace\),

\[\forall x \in\mathbb{R}\backslash\lbrace{-4}\rbrace,\;\;\;g'(x)=\frac{62}{(2x+8)^2}\]
