\(h\) est donc dérivable sur \(\mathbb{R}\backslash\lbrace{\frac{1}{2}}\rbrace\) et pour tout \(x\in\mathbb{R}\backslash\lbrace{\frac{1}{2}}\rbrace\),

\(h'(x)=\frac{9(6x-3) -6(9x-1)}{(6x-3)^2}\)
Ainsi :
\[\forall x \in\mathbb{R}\backslash\lbrace{\frac{1}{2}}\rbrace,\;\;\;h'(x)=\frac{-21}{(6x-3)^2}\]
