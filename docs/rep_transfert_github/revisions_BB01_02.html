La fonction \(f\) est un quotient de fonctions définies et dérivables sur \(\mathbb{R}\backslash\lbrace{\frac{2}{3}}\rbrace\) de plus le dénominteur ne s'annule pas sur \(\mathbb{R}\backslash\lbrace{\frac{2}{3}}\rbrace\).
\(f\) est donc dérivable sur \(\mathbb{R}\backslash\lbrace{\frac{2}{3}}\rbrace\) et pour tout \(x\in\mathbb{R}\backslash\lbrace{\frac{2}{3}}\rbrace\),

\(f'(x)=\frac{7(9x-6) -9(7x+10)}{(9x-6)^2}\)
Ainsi :
\[\forall x \in\mathbb{R}\backslash\lbrace{\frac{2}{3}}\rbrace,\;\;\;f'(x)=\frac{-132}{(9x-6)^2}\]
