\(h\) est donc dérivable sur \(\mathbb{R}\backslash\lbrace{2}\rbrace\) et pour tout \(x\in\mathbb{R}\backslash\lbrace{2}\rbrace\),

\(h'(x)=\frac{-7(3x-6) -3(-7x+9)}{(3x-6)^2}\)
Ainsi :
\[\forall x \in\mathbb{R}\backslash\lbrace{2}\rbrace,\;\;\;h'(x)=\frac{15}{(3x-6)^2}\]
